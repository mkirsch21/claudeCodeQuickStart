# CLAUDE.md

## Project Purpose

This repository helps beginners quickly set up and use Claude Code with a fully free development stack.

It provides:
- **Multi-platform setup** - Works on macOS, Linux, and Windows
- **Docker support** - Reproducible development environment
- **Working examples** - Real Claude API integration samples
- **Zero cost** - Uses only free tools and free API tier

## Coding Standards

- Keep scripts simple and well-commented
- Use beginner-friendly explanations
- Avoid unnecessary dependencies
- Test on multiple platforms
- Include error handling and helpful messages
- Provide links to documentation

## Architecture

- `setup/` - Platform-specific installation scripts
- `templates/` - Example Claude API usage
- `.devcontainer/` - VS Code dev container configuration
- `docker-compose.yml` - Docker setup for isolated environments

## Contributing

When adding new features:
1. Keep it beginner-friendly
2. Test on multiple OS platforms
3. Update documentation
4. Use free tools and services only
